## Syfte

Detta kodarkiv innehåller alla centrala Python- och dokumentationsfiler som används för att **verifiera, simulera och dokumentera RP9-modellen** – en teoretisk och praktisk modell för koherent energiöverföring.

Varje kodfil har en egen **förklaringsfil** (`.md`) som beskriver syfte, funktion och resultat.  
Projektet är organiserat för att möjliggöra både **matematisk validering** och **experimentell reproducerbarhet**.
